whipcode


A low cost API for running untrusted code securely on remote servers.

Currently in open beta.

Subscribe to a free or paid plan on RapidAPI to get started.


Available interpreters are Python, Node.js, Bash, Ruby, Perl and Lua
(With more to come!)


We do not store submitted code or any form of user data on our servers.

Billing, authentication and ratelimits are managed by RapidAPI.
